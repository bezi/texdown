Database logic
==============

User
----
### Name
Human readable name, e.g. "Oscar"
### Email
Email address for logging in
### Google OAuth token

### Files
array of file ids this person owns

File
----
### FileName
File name
### Text
Contents of file
### Tags
array of tags
### Owner
owner id
### Permissions
'R', 'RW', or 'P' for read, read/write, or private
